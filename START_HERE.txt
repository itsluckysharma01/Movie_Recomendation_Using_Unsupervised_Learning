   📁 start_server.bat

Then open your browser to:
   
   🌐 http://localhost:5000

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Open Command Prompt in this folder

Step 2: Install dependencies
   
   pip install -r requirements.txt

Step 3: Start the server
   
   python app.py

Step 4: Open browser to
   
   http://localhost:5000

🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: "Port 5000 already in use"
Solution: Kill the process or change port in app.py

Problem: "Module not found"
Solution: pip install -r requirements.txt

Problem: "Movie not found"
Solution: Try popular movies like "Matrix" or "Inception"

Problem: Page not loading
Solution: Make sure server is running at localhost:5000
